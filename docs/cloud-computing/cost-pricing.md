# Cost & Pricing

## Pay-As-You-Go
Charged for actual usage (hours, GB, requests).

## Common Cost Areas
- Compute (VM hours)
- Storage (GB/month)
- Data transfer (egress)
- Managed services (databases, queues)

## Saving Tips
- Right-size resources
- Use auto scaling
- Turn off idle dev servers
- Reserved / spot instances for steady workloads
