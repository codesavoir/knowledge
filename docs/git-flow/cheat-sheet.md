# Cheat Sheet

## Everyday Commands
```
git status
git add file
git commit -m "Message"
git pull --rebase
```

## Branching
```
git checkout -b feature/login
```

## Stash
```
git stash
```

## Undo Last Commit (keep changes)
```
git reset --soft HEAD~1
```

## Show Log Graph
```
git log --oneline --graph --decorate --all
```
